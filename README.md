# WhatsApp OSINT Toolkit

A modular toolkit for parsing WhatsApp exports into structured evidence packages.  
Built for digital forensics, OSINT workflows, and transparent stakeholder reporting.

---

## Features
- Auto-detection of WhatsApp export formats
- Evidence packaging into CSV, JSON, and GeoJSON
- Deleted message flags and keyword highlights
- Media indexing for images, audio, and video
- Executive summaries with per-author statistics
- Secure zipped archives with timestamped filenames
